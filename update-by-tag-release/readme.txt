Start
1. in gitkraken
2. clic right in branch pixL-master "create tag here"
3. name tag whith the last version check changelog.md
4. push this tag
5. clic right in the tag
6. push vx.xxx.x to origin
7. in your pc zip the folder "shinretro"
8. go to https://github.com/pixl-os/shinretro/releases/new
9. choose a tag create in step 3
10. clic on "Generate release notes"
11. clic on "Set as a pre-release"
12. upload in "Attach binaries by dropping them here or selecting them." the files present into "update-by-tag-release" without readme.text and the shinretro.zip
13. clic on "Publish release"
Done
